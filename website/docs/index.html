Features

Everything images, in one place

Image Inspector

Dimensions, file size, MIME type, EXIF/GPS data, ICC profiles, alt text, DPI — all from a right-click.

Format Converter

Real pixel-level conversion between PNG, JPEG, WebP, AVIF, TIFF, BMP, QOI, ICO, SVG and more. Not just renaming.

QR Code Studio

Generate QR codes for URLs, WiFi, vCards. Read QR codes from any image on the page. Export as PNG or SVG.

Smart Downloader

Bulk download with filters, smart filenames, format conversion on-the-fly, ZIP export. Finds originals, not thumbnails.

Quick Editor

Resize, crop, rotate, flip, adjust brightness/contrast, apply filters — all without leaving the browser.

Color Extraction

Pick any pixel color, extract dominant palettes, copy as HEX/RGB/HSL. Perfect for designers.

SVG Tools

Render SVG to raster at any resolution, inspect structure, view source. No other extension does this.

Clipboard Magic

Copy any image as PNG to clipboard, copy as base64, paste images into the editor from clipboard.

Accessibility Audit

Find missing alt text, oversized images, broken images, and performance issues on any page.

How It Works

Powered by Rust, runs in your browser

R

Read

Detects every image on the page — <img>, CSS backgrounds, canvas, SVG, base64 — and reads all metadata instantly.

O

Optimise

Rust/WASM engine processes images at native speed — convert formats, resize, crop, adjust, generate QR — all offline.

O

Output

Save to disk, copy to clipboard, download as ZIP, export as any format — your images never leave your machine.

Under the hood

Rust WebAssembly Manifest V3 Tailwind CSS image-rs resvg qrcode kamadak-exif

Your images never leave your device

Every operation runs locally in WebAssembly. No servers. No uploads. No telemetry. No ads. No affiliate links. We don't even know you exist.

Zero network calls

WASM module has no network access. Processing is physically impossible to leak.

Minimal permissions

Only activeTab, contextMenus, downloads, storage, sidePanel. No browsing history. No <all_urls>.

Open source

Full source on GitHub. Reproducible builds. Verify the WASM binary matches the code yourself.
